Help file
python licor_summary.py <<output directory>> <<start date>> <<end date>>

output directory: 
	The directory where the csv files will be stored.

start date & end date: 
	The dates should be in a YYYY-MM-DD format. 
	If the dates are not included the script downloads all the data available.
	If the start date is the only one included then the script downloads all 
	the data from start date to today.
